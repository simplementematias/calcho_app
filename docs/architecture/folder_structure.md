Convenciones:
- Cada feature encapsula `domain`, `data` y `presentation`.
- `domain` nunca depende de Firebase ni de Flutter UI.
- `presentation/providers` usa Riverpod con code generation (`@riverpod`).
- Integración Firebase y DTOs quedan en `data`.
